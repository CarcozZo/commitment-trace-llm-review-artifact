# FAIR Checklist

## Findable

- [x] Dataset title is stable: CommitmentTrace-LLM.
- [x] Version is specified for the anonymous review artifact.
- [x] File manifest and checksums are provided.
- [ ] Public DOI will be added after acceptance.

## Accessible

- [x] Review artifact can be shared through an anonymous repository.
- [x] Data files use open formats: JSONL, JSON, CSV, Markdown.
- [x] Large JSONL files are gzip-compressed.
- [ ] Public long-term repository will be selected after acceptance.

## Interoperable

- [x] Machine-readable schema is included.
- [x] Splits are explicit and machine-readable.
- [x] Action labels and policy labels are documented.
- [x] Online-observation and hidden-field boundaries are documented.

## Reusable

- [x] Dataset card documents intended use and non-goals.
- [x] Baseline protocol documents fairness rules.
- [x] Validation script checks row counts and leakage boundaries.
- [x] Summary script reconstructs key reported test-set tradeoffs.
- [ ] Final licence must be confirmed before public release.

## Blocking Issues Before Public Release

- Confirm final data and code licences.
- Replace anonymous placeholders with real creator metadata.
- Deposit a DOI-backed archive or equivalent persistent repository record.
- Verify that the final public record and manuscript statements match exactly.
